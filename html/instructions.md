# Create a Travel Agency Page

## Requirements

- Include a `<!DOCTYPE html>` declaration.
- Include an `html` element with `lang="en"`.
- Include a `head` element containing a meta void element with `charset="utf-8"` and a `title` with the text "Travel Agency Page".
- Include a `meta` tag in the `head` element that contains a short description of the website for SEO.
- Include an `h1` element to present the travel destinations.
- Include a paragraph below the `h1` element introducing the travel opportunities.
- Include an `h2` element with the text "Packages".
- Include a `p` element briefly introducing the various packages.
- Include an unordered list with two list items. The two list items should have the text "Group Travels" and "Private Tours", respectively. The text of each list item should be enclosed by an anchor element.
- Include an `h2` element with the text "Top Itineraries".
- Include at least three `figure` elements, each containing an anchor element and a `figcaption` element.
- The three anchor elements should contain an `img` element with an appropriate `alt` attribute and a `src` attribute set to a valid image. You can use `https://cdn.freecodecamp.org/curriculum/labs/colosseo.jpg`, `https://cdn.freecodecamp.org/curriculum/labs/alps.jpg`, and `https://cdn.freecodecamp.org/curriculum/labs/sea.jpg`.
- All five anchor elements should have an `href` attribute with the value `https://www.freecodecamp.org/learn` and a `target` attribute with the value `_blank`.
